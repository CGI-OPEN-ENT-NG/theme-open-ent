La première connexion à Colibri comprend une étape préalable indispensable, qui vous permet de personnaliser vos informations de connexion.

  • De quoi se munir ?

    Pour se connecter à Colibri, vous devez vous munir de votre adresse électronique personnelle.

  • Comment accéder à Colibri ?

    Vous pouvez accéder à Colibri en passant par la page d’authentification du projet
    https://colibri.ac-martinique.fr

  • Comment se connecter la première fois ?

    Utiliser les identifiants ci-dessous sur la page de connexion de Colibri :
    • Saisissez votre identifiant
    • et le code d'activation
    • Cliquez sur Connexion
    {{login}}
    {{activationCode}}

Pour sécuriser vos accès, il vous sera demandé de choisir un nouveau mot de passe et, si vous le souhaitez, d’indiquer votre courriel personnel pour faciliter la récupération du mot de passe en cas d’oubli, et de saisir un numéro de téléphone portable. Vous aurez aussi la possibilité de consulter la charte de Colibri dont vous devrez accepter les termes.

Votre nouveau mot de passe doit contenir au minimum 6 caractères.

Désormais, vous pouvez vous connecter à Colibri en utilisant votre nouveau mot de passe.
